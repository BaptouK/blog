# blog.baptouk.live

Mon blog personnel, généré avec [Zola](https://www.getzola.org/) et le thème [tabi](https://github.com/welpo/tabi).

Disponible sur [blog.baptouk.live/]( blog.baptouk.live/)

## Stack

- **Zola** 0.22.1 — générateur de site statique
- **tabi** — thème Zola

## Développement local

```bash
zola serve
```

Le site est accessible sur `http://127.0.0.1:1111`.

## Build

```bash
zola build
```

Le résultat est généré dans `public/`.
