## Projet 

Pokérixe est un site de combat de Pokémons tour par tour.

Le projet est développé en 2 étapes : La première c'est le Front-end, que je développe seul depuis le début de l'année 2026. <br>
La deuxième partie c'est le Back-end, qui est le sujet du projet de fin d'année de ma deuxième année d'étude qui se dérouleras la première semaine de Mai 2026.

## Les technologies


### Le Front-end 

Le lien du [github](https://github.com/BaptouK/PokeRixe) <br>
Le lien pour [tester](https://pokerixe.baptouk.live/)

J'utilise le framework Angular pour une meilleur avoir toutes les fonctionnalités néccessaires pour faire un site qui bouge, avec gestions d'évenement etc... <br>

Il y a déja des fonctionnalités disponibles, comme le pokédex des 150 premiers pokémons avec filtres par types, affichages des données d'un pokémon, connexions mocké, gestion d'une équipe, combat mocké, rejoindre/créer une game mocké.
